## Props

| Prop          | Type                    | Default      | Description                           |
| ------------- | ----------------------- | ------------ | ------------------------------------- |
| `title`       | `string`                | required     | Main headline                         |
| `subtitle`    | `string`                | —            | Smaller text below title (retro font) |
| `description` | `string`                | —            | Body text below subtitle              |
| `badges`      | `HeroBadge[]`           | `[]`         | Mini badge pills above the title      |
| `actions`     | `HeroAction[]`          | `[]`         | CTA buttons                           |
| `variant`     | `"centered" \| "split"` | `"centered"` | Layout variant                        |
| `className`   | `string`                | —            | Additional classes                    |
