## Usage

---

```tsx
import Hero2 from "@/components/ui/8bit/blocks/hero2";

<Hero2
  title="SHIP FASTER"
  subtitle="Retro components for the modern web"
  description="Drop-in 8-bit styled components."
  actions={[
    { label: "START BUILDING", variant: "default" },
  ]}
/>
```

### With custom visual

```tsx
<Hero2
  title="SHIP FASTER"
  visual={<img src="/screenshot.png" alt="Preview" className="pixelated" />}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | required | Main headline |
| `subtitle` | `string` | — | Smaller text below title |
| `description` | `string` | — | Body text |
| `badges` | `HeroBadge[]` | `[]` | Badge pills above the title |
| `actions` | `HeroAction[]` | `[]` | CTA buttons |
| `visual` | `ReactNode` | Placeholder | Right-side visual content |
| `children` | `ReactNode` | — | Extra content below actions |
| `className` | `string` | — | Additional classes |
