## Props

| Prop          | Type           | Default  | Description                 |
| ------------- | -------------- | -------- | --------------------------- |
| `title`       | `string`       | required | Main headline               |
| `subtitle`    | `string`       | —        | Smaller text below title    |
| `description` | `string`       | —        | Body text                   |
| `badges`      | `HeroBadge[]`  | `[]`     | Badge pills above the title |
| `actions`     | `HeroAction[]` | `[]`     | CTA buttons                 |
| `children`    | `ReactNode`    | —        | Extra content below actions |
| `className`   | `string`       | —        | Additional classes          |
