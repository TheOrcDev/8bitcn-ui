## Usage

---

```tsx
import Feature3 from "@/components/ui/8bit/blocks/feature3";

<Feature3
  title="Built Different"
  items={[
    { icon: ">>", title: "Lightning Fast", description: "...", size: "wide" },
    { icon: "{}", title: "Type Safe", description: "..." },
    { icon: "##", title: "Accessible", description: "..." },
    { icon: "</>", title: "Copy Paste", description: "...", size: "tall" },
    { icon: "~~", title: "Themeable", description: "...", badge: "NEW" },
    { icon: "[]", title: "Composable", description: "...", size: "wide" },
  ]}
/>
```

## Card Sizes

- `"default"` — 1 column, 1 row
- `"wide"` — spans 2 columns
- `"tall"` — spans 2 rows

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"Built Different"` | Section heading |
| `description` | `string` | — | Section subheading |
| `items` | `BentoItem[]` | 6 defaults | Grid items |
| `className` | `string` | — | Additional classes |

### BentoItem

| Prop | Type | Description |
|------|------|-------------|
| `icon` | `ReactNode` | Icon or symbol |
| `title` | `string` | Feature name |
| `description` | `string` | Feature description |
| `size` | `"default" \| "wide" \| "tall"` | Card size in grid |
| `badge` | `string` | Optional badge text |
| `className` | `string` | Additional classes per card |
