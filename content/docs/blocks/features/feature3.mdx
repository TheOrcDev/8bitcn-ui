## Usage

---

```tsx
import Feature3 from "@/components/ui/8bit/blocks/feature3";

<Feature3
  title="The Adventure Awaits"
  items={[
    { icon: "01", title: "Choose Your Class", description: "..." },
    { icon: "02", title: "Enter the Dungeon", description: "...", badge: "FAN FAVORITE" },
    { icon: "03", title: "Defeat Bosses", description: "..." },
    { icon: "04", title: "Upgrade Your Gear", description: "..." },
    { icon: "05", title: "Climb the Ranks", description: "...", badge: "COMPETITIVE" },
  ]}
/>
```

Carousel loops infinitely and shows 1 card on mobile, 2 on tablet, 3 on desktop.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"The Adventure Awaits"` | Section heading |
| `description` | `string` | — | Section subheading |
| `items` | `CarouselFeature[]` | 5 defaults | Carousel cards |
| `className` | `string` | — | Additional classes |

### CarouselFeature

| Prop | Type | Description |
|------|------|-------------|
| `icon` | `ReactNode` | Icon or number |
| `title` | `string` | Feature name |
| `description` | `string` | Feature description |
| `badge` | `string` | Optional badge text |
